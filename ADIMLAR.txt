📍 ADIM 1: GitHub'a Yükle (2 dakika)
═══════════════════════════════════════════════════════════════

Terminal'de sırayla:

1️⃣  chmod +x deploy-setup.sh
2️⃣  ./deploy-setup.sh

3️⃣  Browser'da aç: https://github.com/new
    - Repository name: freelance-platform
    - Public seç
    - Create repository tıkla

4️⃣  Terminal'de:
    git remote add origin https://github.com/newmekintos/freelance-platform.git
    git push -u origin main

✅ GitHub'a yüklendi!

═══════════════════════════════════════════════════════════════

📍 ADIM 2: Railway'de Deploy (5 dakika)
═══════════════════════════════════════════════════════════════

1️⃣  https://railway.app → Login with GitHub

2️⃣  New Project → Deploy from GitHub repo → freelance-platform seç

3️⃣  BACKEND SERVİSİ EKLE:
    Add Service → Empty Service
    Settings:
    - Name: freelance-backend
    - Root Directory: backend
    - Start Command: npm start
    
    Variables ekle:
    JWT_SECRET=freelance-super-secret-key-2024-xyz
    NODE_ENV=production
    
    Deploy tıkla

4️⃣  FRONTEND SERVİSİ EKLE:
    Add Service → Empty Service
    Settings:
    - Name: freelance-frontend
    - Root Directory: frontend
    - Build Command: npm run build
    - Start Command: npm run start
    
    Variables ekle:
    VITE_API_URL=https://freelance-backend.up.railway.app/api
    (Backend URL'ini kopyala yapıştır)
    
    Deploy tıkla

5️⃣  CORS AYARI:
    Backend servisine dön → Variables → Add:
    CORS_ORIGIN=https://freelance-frontend.up.railway.app
    (Frontend URL'ini kopyala yapıştır)
    
    Redeploy tıkla

✅ Deploy tamamlandı!
